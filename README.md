# node-express

## Frankenstein Express demo

Simple Node.js 22 LTS Express app that responds with a Frankenstein-themed message: "It's alive!".

### Prerequisites

- Node.js 22 LTS or later

### Install

```bash
npm install
```

### Run the lab server

```bash
npm start
```

By default the server listens on `http://localhost:3000`.

### Try it out

- Open `http://localhost:3000` in your browser to see the "It's alive!" page.
- Call `http://localhost:3000/status` to get a small JSON payload with the creature's vital signs.
